# Microokoa Guaranty Capital Website

This project is a professional, responsive microfinance website designed for Microokoa Guaranty Capital, inspired and benchmarked against progressivecr.com with focus on clean UI, trust elements, donation-loan mechanics, and Kenyan market context.

## Features
- Responsive layout adapting from 320px mobile to 1920px desktop
- Fixed header with logo, navigation with dropdowns, and prominent CTA button
- Interactive loan calculator with 10x multiplier
- Image slider with parallax and overlay
- Donation-based loan explanation and community impact emphasis
- Testimonials carousel and trust badges
- Digital wallet integration display (M-Pesa, Airtel Money)
- Footer with comprehensive info, social media links, and contact details
- Accessibility features (ARIA, keyboard navigability)
- Semantic HTML5 structure and modern CSS Grid/Flexbox layouts
- Progressive Web App manifest included

## Setup Instructions
1. Replace placeholder images (logo SVG, favicon PNG, slides, icons) in the `images/` folder.
2. Deploy on any static hosting environment or integrate into your backend.
3. Ensure JS file `script.js` and CSS `styles.css` are loaded with the HTML.
4. Customize links and text content as needed.

## Development Notes
- Loan calculator sliders update loan access in real-time.
- Hamburger menu for mobile toggles nav visibility.
- Dropdown menus adapt on desktop and mobile for usability.
- Testimonials auto-rotate but support manual navigation.
