# Microokoa Guaranty Capital Website

A professional, responsive microfinance website for Microokoa Guaranty Capital featuring a unique donation-based financing model.

## Features

### Core Functionality
- **Dynamic Single-Page Application**: Smooth navigation between sections without page reloads
- **Interactive Loan Calculator**: Real-time calculation of loan access based on donations
- **Hero Image Slider**: Auto-rotating slides with manual controls
- **Testimonial Carousel**: Customer success stories with navigation
- **Responsive Design**: Mobile-first approach (320px to 1920px screens)
- **Accessibility**: ARIA labels, keyboard navigation, semantic HTML

### Sections Included
1. **Home**: Hero slider, loan calculator, how it works, digital wallets, testimonials
2. **About**: Company mission, values, statistics
3. **Our Products**: QuickLoans, BusinessLoans, Donate & Access programs
4. **Borrow**: Loan eligibility checker, application process
5. **Donate**: Donation calculator, payment methods, impact tracking
6. **Contact Us**: Contact form, support channels, social media
7. **Careers**: Job listings, application process
8. **FAQs**: Categorized frequently asked questions

### Technical Features
- **CSS Grid/Flexbox**: Responsive layouts with 12-column grid on desktop
- **JavaScript Modules**: Organized code with separation of concerns
- **Form Validation**: Client-side validation for all forms
- **Performance Optimized**: Lazy loading, optimized images
- **Cross-browser Compatible**: Works on all modern browsers
